Malware

El software malicioso abarca el conjunto de software elaborado con fines malintencionados: virus, troyanos, gusanos, spyware, etc.

Virus

Se trata de un software creado y desarrollado con el ánimo de alterar un equipo de forma malintencionada. El virus se puede implantar en el sistema de múltiples formas, pero todos tienen un fin “malicioso” o “maligno”: controlar, espiar, alterar, sustraer, eliminar, etc. Por tanto, se pueden representar de muchas maneras diferentes.
